# Sales Forecasting Using Machine Learning

## Project Description

This project implements a simple Sales Forecasting system using the Linear Regression Machine Learning algorithm. Historical monthly sales data is used to train a predictive model that forecasts future sales values. The project demonstrates the basic workflow of Machine Learning including data preparation, model training, evaluation, prediction, and visualization.

## Machine Learning Model

### Linear Regression

Linear Regression is used to identify the relationship between months and sales values. The model learns from historical data and predicts future sales trends.

---

## Project Workflow

1. Import required libraries.
2. Create sales dataset.
3. Convert data into a Pandas DataFrame.
4. Split dataset into training and testing sets.
5. Train Linear Regression model.
6. Predict test data values.
7. Calculate Mean Absolute Error (MAE).
8. Forecast future sales for Months 13, 14, and 15.
9. Visualize results using Matplotlib.
